# val_log_reader
automatizacion de analisis de logs de validador vl550
Comando en terminal: python log_analyzer.py logs/validator* lista_eventos_vl550.xlsx reportes_html/archivo_eventos*.html; Test-Path "reportes_html/archivo_eventos*.html"
Ejemplo de comando en terminal: python log_analyzer.py logs/validator.2 lista_eventos_vl550.xlsx reportes_html/archivo_eventos_02.html; Test-Path "reportes_html/archivo_eventos_02.html
